# Image Classification Project with Discord Bot

Hello! This project is a Discord bot that utilizes a trained model for image classification. Read below to learn more.

## Description

This bot is designed for image classification using a pre-trained model. It allows users to send images and receive predictions from the model in response. The bot is implemented using the Discord API and the discord.py library.

## Usage

1. Invite the bot to your Discord server.
2. Use the `$hello` command to greet the bot.
3. Use the `$heh` command to repeat the word "heh" a specified number of times.
4. Use the `$check` command with an attached image for classification.
